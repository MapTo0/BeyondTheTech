# Beyond The Tech
#### Simple Blogging System for Tech news

Beyond The Tech is a simple blogging system developed in ruby.

## End points

### GET
 - "/home"
 - "/login/view"
 - "/register/view"
 - "/posts?query"
 - "/posts/view"
 - "/posts/\__id__/"
 - "/posts/\__id__/view"

### POST
 - "/login"
 - "/register"
 - "/posts"

### PUT
 - "/posts/\__id__/"

Powered by Sinatra
